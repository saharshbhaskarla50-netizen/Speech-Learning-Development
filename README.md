# Speech and Leadership Development (SLD)

An AI-powered public speaking coach that analyzes 
your speech and helps you improve over time.

→ GPT Feedback → Dashboard

- Backend: Next.js API routes
- AI: OpenAI (Whisper + GPT-4o mini)
- Database: Supabase
- Hosting: Vercel
